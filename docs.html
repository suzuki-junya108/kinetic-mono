RULES 05

DO
Uppercase Micro-Labels for all metadata
Asymmetric layouts, 70% hero title widths
Use Yellow as color field on dark or black
Background shifts for visual sectioning
Kinetic Hover: color snap + translateX(4px)
✗ DON'T
Use border-radius (pills, rounded corners)
Yellow text on white/gray backgrounds
1px borders for layout containment
Gradients for depth (use surface shifts)
Circular progress rings (use horizontal bars)

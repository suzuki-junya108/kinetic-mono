構成ルール
  • viewBox: 0 0 24 24、1px内側セーフゾーン
  • stroke-width: 1.5（デフォルト）/ 2（強調）
  • stroke-linecap: square（丸は使わない）
  • stroke-linejoin: miter（丸は使わない）
  • fill: none（デフォルト）、currentColor（ステータスのみ）
  • 全座標は整数または0.5グリッド上
  • 角度: 0°、45°、90° のみ
  • 構造アイコンに2pxノッチ（右上）
使用方法

SVGスプライトシステムを使用。以下のコードでアイコンを表示できます。

<!-- Basic usage --> <svg class="km-icon"> <use href="#km-icon-arrow-right"></use> </svg> <!-- With size variant --> <svg class="km-icon km-icon--lg"> <use href="#km-icon-search"></use> </svg> <!-- Filled variant (status dots) --> <svg class="km-icon km-icon--fill" style="color:var(--a1)"> <use href="#km-icon-dot"></use> </svg> <!-- JavaScript API --> KineticMono.icon('arrow-right', { size: 'lg' })
